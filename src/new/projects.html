Projects

ringOS (archived)

ringOS is small operating system. Development started in September 2021. Then in June 2022 we switched to Limine boot protocol and starting from scratch. Also, main ringOS goal is to have nice own GUI. Archived because of project complexity.

Termi

Termi is a powerful terminal written in C++ using OpenGL and Dear ImGui. The goal is to create independet terminal which use OpenGL and Dear ImGui to render and has own commands, commands are not from executables from host operating system. Windows version is less synced and also Windows version has GUI launcher/updater written in C# and XAML.

Termi-Lua

Termi-Lua is Lua version of Termi. It doesn't use OpenGL, Dear ImGui or any other graphics library Because it doesn't use graphics library and it is written in Lua it is very easy to port.

Cmbal

Small programming language written in C. It is mbal clone.
